# Doc

用于存放正式系统 `qts/` 的补充文档。

要求：

- 只写 `qts/` 文件的总括。
- 与 `qts/` 目录保持同步，尤其是 `qts/core/` 与 `qts/infra/` 的边界。
- 不记录 `lab/` 的实验细节。
- 默认配置见 `configs/qts.yaml`。
- 入口编排见 `qts/infra/entrypoints.py`，统一报表与信号规范见 `qts/infra/report.py`。
- 正式入口配置可使用 `configs/backtest.yaml`、`configs/close_report.yaml`、`configs/stock_selection.yaml`。
